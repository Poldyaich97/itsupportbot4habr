# IT Support Bot (fork)

Телеграм‑бот для поддержки с форумными тредами

## Что умеет

- Личные сообщения пользователей улетают в отдельные треды в супергруппе, ответы из треда уходят обратно.
- Название треда берётся из текста первой реплики.
- Дежурные: панель с добавлением/удалением, самовключение/выход, упоминание дежурных при новой заявке.
- Закрытие заявки кнопкой или командой: тред закрывается (🔒), у пользователя запрашивается оценка 1/3/5, результат пишется в тред.
- Статистика в SQLite: общее/открытые/закрытые, средняя оценка, топ пользователей; автопубликация в выделенный тред после каждого закрытия и по команде.
- Админ‑действия : сброс статистики/заявок, удаление всех тредов заявок (кроме статистики).
- Секреты из `.env`: `TELEGRAM_TOKEN`, `TELEGRAM_SUPPORT_CHAT_ID`. БД `support.db` создаётся автоматически.

## Быстрый старт

```bash
pip install "aiogram>=3,<4"
python3 main.py
```

Требования в Telegram: супергруппа с включёнными темами, бот — админ с правом создавать/удалять темы.

## Настройка окружения

Создайте файл `.env` (не коммитить):

```
TELEGRAM_TOKEN=your_bot_token
TELEGRAM_SUPPORT_CHAT_ID=-100xxxxxxxxxx
```
